Code for finding min and max

Write a program in any language to find the maximum and minimum element in an array.

int array = list(map(int,input("enter elements of array").split()))
int maxNumber=max(array)
int minNumber=min(array)
print("max :"+str(maxNumber))
print("min :"+str(minNumber))



sample input : enter elements of array : 1 2 3
sample Output : Max: 3 , Min : 1
Variable used : maxNumber (int),minNumber(int) ,array(list)
time complexity: O(n)
